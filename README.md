# EngToolbox Resources

Open engineering resources from [EngToolbox.org](https://engtoolbox.org) — code examples, validation data, templates, and reference files.

## What's here

| Folder | Contents |
|---|---|
| `/openfoam` | OpenFOAM case files and tutorials |
| `/fds` | FDS input files and validation cases |
| `/python` | Python scripts for engineering calculations |
| `/validation` | Test case data for calculator verification |
| `/templates` | Engineering report and calculation templates |

## About EngToolbox

EngToolbox is a multi-discipline engineering platform with interactive calculators, reference data, and learning resources for fire, CFD, and general engineering.

- 🔥 [Fire Box](https://engtoolbox.org/firebox/) — Fire safety engineering tools
- 🌊 [CFD Box](https://engtoolbox.org/cfdbox/) — Computational fluid dynamics tools
- 🔧 [Eng Box](https://engtoolbox.org/engbox/) — Cross-discipline engineering reference
